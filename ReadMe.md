# Study Notes API
A simple background API built with
- Node.js
- express
- TypeScript
- MongoDB
- Mongoose
- JWT authentication
## Features
- Register
- Login
## Installation
1. Clone project
2. Install Repository
3. Create .env file
- PORT=5000
- MONGO_URL=your_mongodb_connection_string
- JWT_SECRET=your_secret
4. Run Project
- npm run dev
